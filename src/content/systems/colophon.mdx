# Colophon

This site was built with:

- **Framework:** Next.js (App Router)
- **Language:** TypeScript (strict mode)
- **Styling:** Tailwind CSS
- **3D/Experience:** React Three Fiber, GSAP, Lenis
- **Content:** MDX, Velite
- **State:** Zustand

## Typography

- **Headings:** Averia Serif Libre
- **Body:** Inter

## Color System

- **Ink Black:** `#13161F`
- **Parchment:** `#FAF6F0`
- **Bright Indigo:** `#0036D8`
- **Pacific Cyan:** `#3A7D8C`

## Design Principles

1. Experience leads, utility follows
2. Discipline over spectacle
3. Systems with memory
4. Human content inside structured containers
